URL Redirector

Any URL on any of the domains listed at the bottom of this page (other than this home page) will redirect to bit.ly (a URL shortener) with the last seven characters of the URL being the ID of the shortened bit.ly URL. Examples:

  • zooom.us.to/j/2K3lzgE → bit.ly/2K3lzgE (ID = 2K3lzgE)
  • tikok.us.to/@2K3lzgE → bit.ly/2K3lzgE (ID = 2K3lzgE)
  • instagam.us.to/p/B2K3lzgE → bit.ly/2K3lzgE (ID = 2K3lzgE)
