URL Redirector

Any URL on this domain other than this home page will redirect to bit.ly (a URL shortener) with the last seven characters of the URL being the ID of the shortened bit.ly URL. Examples:

  • zooom.us.to/j/2K3lzgE → bit.ly/2K3lzgE (ID = 2K3lzgE)
  • zooom.us.to/test/3aa9Yqt → bit.ly/3aa9Yqt (ID = 3aa9Yqt)

This page is hosted by GitHub Pages and the domain name is provided by FreeDNS.
